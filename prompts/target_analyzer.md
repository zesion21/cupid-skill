# 对方行为分析器

## 任务

从原材料中提取对方的行为模式、性格特征和态度信号，构建 Target Profile（对方画像）。

**原则**：只从原材料中提取有依据的信息，没有依据就标注"原材料不足"。不过度脑补。

---

## 提取维度

### 1. 消息节奏分析（聊天记录）

从聊天记录中识别：

- **回复速度模式**
  - 秒回型：通常几分钟内回复
  - 慢回型：经常几小时甚至隔天回复
  - 不稳定型：有时快有时慢，无规律
  - 已读不回型：经常不回复或很晚回复

- **主动发起频率**
  - 高频主动：经常主动找你
  - 低频主动：偶尔主动
  - 被动型：几乎从不主动，但会回复
  - 不回应型：连回复都很敷衍

- **消息长度趋势**
  - 长句型：经常发长消息
  - 短句型：回复简短
  - 不稳定：有时长有时短
  - 递减型：越来越短（可能信号）
  - 递增型：越来越长（可能信号）

```
输出格式：
回复模式：[秒回/慢回/不稳定/已读不回]
主动频率：[高频/低频/被动/不回应]
消息长度：[趋势描述]
```

### 2. 内容模式分析

- **话题偏好**
  - 日常分享型：喜欢分享生活琐事
  - 工作导向型：聊工作相关居多
  - 深度对话型：会聊价值观、情感
  - 轻松娱乐型：聊游戏、电影、八卦
  - 话题回避型：某些话题刻意回避

- **表达方式**
  - 直接型：想说什么直接说
  - 暗示型：喜欢用间接表达
  - 回避型：遇到关键问题就绕开
  - 试探型：喜欢先试探再表态

- **分享欲程度**
  - 高分享欲：主动分享日常、照片、心情
  - 低分享欲：很少分享个人生活
  - 选择性分享：只分享某些类型的内容

```
输出格式：
话题偏好：[类型]
表达方式：[类型]
分享欲：[高/低/选择性]
```

### 3. 关系信号分析

识别对方对用户的态度信号：

**友善信号**：
- 主动找你聊天
- 分享日常/心情
- 关心你的状态（问你最近怎么样）
- 记得你说过的话
- 约你见面/活动

**暧昧信号**：
- 忽冷忽热（主动后又冷淡）
- 刻意保持距离感
- 偶尔越过边界（突然热情）
- 试探性问题（你有没有对象）
- 制造"巧合"接触

**回避信号**：
- 回复敷衍（哦/嗯/好的）
- 话题回避（聊到关键问题就绕开）
- 刻意疏远（减少互动频率）
- 拒绝邀约（找理由推脱）

**不确定信号**：
- 情绪波动明显
- 态度不稳定
- 回复模式混乱
- 时好时坏无规律

```
输出格式：
态度信号：[友善/暧昧/回避/不确定]
信号证据：[具体行为举例]
```

### 4. 性格推断（从聊天风格）

从聊天风格推断性格特点：

| 聊天特征 | 可能的性格倾向 |
|---------|---------------|
| 回复快、话题多、表情包多 | 外向、热情、社交型 |
| 回复慢、内容简短、少表情包 | 内向、高冷、务实型 |
| 情绪表达明显、会撒娇/抱怨 | 感性、表达型 |
| 分析式表达、逻辑清晰 | 理性、思考型 |
| 经常确认、问你的感受 | 细心、关注型 |
| 很少表达情绪、话题务实 | 克制、边界型 |

```
输出格式：
性格推断：[标签列表]
推断依据：[聊天特征举例]
```

### 5. 关系定位判断

综合以上分析，判断对方将用户定位在什么位置：

| 定位 | 行为特征 |
|------|---------|
| 朋友 | 友善互动，但保持边界，无暧昧信号 |
| 暧昧对象 | 忽冷忽热，偶尔越界，制造暧昧感 |
| 潜在发展对象 | 主动互动，信号积极，有推进意图 |
| 不感兴趣对象 | 回复敷衍，减少互动，回避邀约 |
| 不确定 | 行为混乱，难以判断 |

```
输出格式：
关系定位：[朋友/暧昧/潜在发展/不感兴趣/不确定]
定位依据：[综合分析]
```

## 输出要求

输出一个 Target Profile 草稿，包含：

1. 消息节奏分析结果
2. 内容模式分析结果
3. 关系信号分析结果
4. 性格推断结果
5. 关系定位判断

每个部分需要有依据（引用原材料中的具体行为），没有依据的维度标注"原材料不足"。

---

## 注意事项

1. **不过度脑补**：分析要有依据，不给虚假希望
2. **接受不确定性**：有些行为确实难以解读，承认这一点
3. **动态视角**：行为可能随时间变化，记录时间线
4. **区分场景**：不同场景（暗恋/暧昧/分手）分析重点不同
